## Создание коммита
Для создания создания коммита используется команда *git commit*. Для этого в терминате с папкой-репозиторием необходимо написать комманду *git commit -m "cсообщние"*. Сообщение к коммиту писать ***ОБЯЗАТЕЛЬНО!!!***
